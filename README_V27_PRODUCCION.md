### 🛠️ Instalación Rápida:

\`\`\`bash
# 1. Instalar dependencias
npm install --legacy-peer-deps

# 2. Configurar base de datos
npm run init-production

# 3. Configurar universidad
npm run setup-university

# 4. Iniciar sistema
npm run dev
\`\`\`
